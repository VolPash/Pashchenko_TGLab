# Класифікація відгуків про фільми (Sentiment Analysis)

## Опис проекту
Цей проект призначений для автоматичної класифікації відгуків про фільми на **позитивні** та **негативні**.  
В якості даних використано набір **IMDB Reviews**, який містить 50 000 відгуків. Мета — навчити модель, яка зможе точно передбачати тон відгуку.

## Кроки виконання

1. **Завантаження даних**
   - Використано набір IMDB з HuggingFace.
   - Комбіновано тренувальний та тестовий набори для подальшого розділення (80/20).

2. **Попередня обробка тексту**
   - Текст переведено у нижній регістр.
   - Видалено пунктуацію та небуквенні символи.
   - Видалено **стоп-слова** англійською.
   - Використано токенізацію простим `split()`.

3. **Векторизація**
   - Текст переведено у **TF-IDF вектори** (максимум 20 000 ознак).
   - Це дозволяє моделі працювати з числовими представленнями тексту.

4. **Навчання моделі**
   - Використано **Logistic Regression**.
   - Розділено дані на тренувальний та тестовий набори (80/20).
   - Використано модель логістичної регресії.

5. **Оцінка якості**
   - Метрики:
     - Accuracy
     - Precision
     - Recall
     - F1-Score
   - Побудовано **confusion matrix** для наочності.

6. **Приклади прогнозів**
   - Виведено 5 випадкових відгуків з передбаченням моделі.
   - Дозволяє перевірити якість роботи моделі на окремих прикладах.

7. **Візуалізація частоти слів**
   - Побудовано топ-20 найпопулярніших слів у позитивних та негативних відгуках.
   - Допомагає зрозуміти ключові слова, що впливають на класифікацію.



## Досягнуті результати
- **Accuracy на тестовому наборі:** ~0.89 
- **F1-Score:** ~0.89  
- Модель коректно передбачає тон більшості відгуків.
